# zendesk-ticket-system
 
## Required
npm v6.14.6
node v12.18.3

## Get start
Go to project directory and run
```
npm i
```
to install node_modules

## Run in production mode
```
npm run start
```

## Build this project
```
npm run build
```

## lint checking
```
npm run lint
```
